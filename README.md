# Heroku 部署機器學習 API
此範例使用鳶尾花朵資料集進行 `XGBoost` 分類器模型訓練。將模型儲存起來，並使用 Flask 建置 API 介面提供輸入值預測。最後並部署到 Heroku 雲端伺服器平台。

## Getting Started
### Fork Project
你可以直接 Fork 此專案到你自己的 GitHub 帳號中，並採用 Heroku 連動 GitHub 專案的方式部署此 API 範例。
